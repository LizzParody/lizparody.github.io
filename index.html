Search Engine

OMG YES! I actually created a Search Engine in Python for the Udacity Intro to CS course.

I learned how to crawl the web, collect all the links in a webpage, and create a index of the links. The index was stored as a dictionary with keyword searches as the key and urls as the value.

I created a lookup function that returns a url given a keyword.

I used a page ranking algorithm to rank the relevancy of pages for each search term.

  • Project in GitHub: Search Engine
  • Date: July 2017

Pioneras Developers

One of the things I'm more passionate about is communities.

I am an organizer of Pioneras Developers, one of the biggest community for girls that are learning to code in Medell√≠n, Colombia.

Diversity leads to better products and results. According to a MIT study diversity in workplaces improves performance, morale and the end product. If there are more women engineers, we can build better teams and better software. Not only that, women who code can be examples to millions of girls to show them programming is something they can do.

We seek to empower women in technology and create a strong community where we can support each other in our learning processes. We provide mentorship, community meetings, and talks on JavaScript, Ruby, HTML, CSS, Git, Angular, Bootstrap, coding challenges, and computer science.

  • Site: Pioneras Dev
